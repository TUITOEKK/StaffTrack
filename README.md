# Overview

StaffTrack helps companies manage field employees in real time.

The system captures employee location, tracks field activity, manages assigned tasks, and syncs data automatically once internet access returns.

Built for logistics teams, sales agents, field officers, technicians, and distributed workforce operations.

---

# Features

## Employee Mobile App

- Real time GPS tracking  
- Live attendance check in and check out  
- Offline first architecture  
- Automatic sync when internet returns  
- Assigned tasks and schedules  
- Route history and movement logs  
- Push notifications  
- Secure Firebase authentication  
- Image upload and field reports   
- Clean and responsive UI  

---

## Admin Dashboard

- Live employee monitoring  
- Task assignment and tracking  
- Attendance analytics  
- Real time map visualization  
- Employee performance reports  
- Activity timeline  
- Offline sync monitoring  
- Department management  
- Role based access control  
- Export reports  
- Notification broadcasting  
- Modern analytics dashboard
